These PNGs are source assets for the embedded ImageEditor custom component
toolbox icon strip. The build packs them into
`build/generated/examples/imageeditor/components/imageeditor_component_icons.c`
with `tools/embed_icon_strip.c`, then links that generated C file into the
`imageeditor_components` plugin.
