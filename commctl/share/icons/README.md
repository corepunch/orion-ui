These PNGs are source assets for the embedded FormEditor/commctl toolbox icon
strip.  The build packs them into `build/generated/commctl/commctl_component_icons.c`
with `tools/embed_icon_strip.c`, then links that generated C file into the
`formeditor_components` plugin.
