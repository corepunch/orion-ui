Create a cohesive set of **64 UI icons** for a UI editor / interface builder toolbox.

---

## Layout

* **Solid white background** (NOT transparent)
* **8×8 grid** (64 icons total)
* Each icon **32×32 px**
* **8 px padding** between icons
* Crisp pixel alignment, clean edges, **no blur**
* All icons centered perfectly inside their cells

---

## Style

* Colorful, vibrant palette that stands out strongly on white
* **Thick black outlines** on every icon (consistent stroke width)
* Bold geometric shapes, strong silhouettes
* Slightly puffy / inflated “toy UI” look, subtly rounded corners
* Chibi-like simplification (large readable forms, no micro detail)
* High contrast, readable at **16×16 scale**
* Subtle gradients and soft highlights (plastic toy-like finish)
* Consistent lighting direction (top-left highlight, bottom-right shadow)
* No text, no labels, no letters
* No realism, no glass, no metallic materials
* Pure iconography for a **toolbox / tool palette**

## Special constraint

* Only a single generic **“button”** icon is included (no states like default/pressed/disabled)
* Each concept is represented once (no duplicates or variants)

---

## Rendering requirements

* Icons must feel like a **cohesive toolbox language system**
* Strong visual consistency across all 64 icons
* Clear silhouettes optimized for small-scale readability
* Uniform stroke thickness and corner radius
* Balanced composition within each 32×32 cell
* No text, no UI screenshots, no realism

## Output
